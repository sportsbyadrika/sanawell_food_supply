# Dew Route Backend (NestJS)

## Run

```bash
npm install
cp .env.example .env
npm run prisma:generate
npm run start:dev
```

## API Modules

- Auth (`/auth/login`)
- Users (`/users`)
- Agencies (`/agencies`)
- Products (`/products`)
- Customers (`/customers`)
- Routes (`/routes`)
- Deliveries (`/deliveries`)
- Billing (`/billing`)
